# crazy-vpn-bot

Requirements: docker

## installing
### Start bot by one command in docker
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/philingood/crazy-vpn-bot/main/start-bot.sh)"
```

### superhard
Clone repo
```bash
git clone https://github.com/philingood/crazy-vpn-bot.git
```
Start bot in docker
